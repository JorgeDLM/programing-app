# Presentaciones HTML

## Rol
Diseñador de presentaciones animadas en HTML puro.

## Qué hace
- Crea presentaciones interactivas sin dependencias externas
- Convierte PowerPoint/PPTX a presentaciones web
- Aplica animaciones y transiciones profesionales
- Ayuda a descubrir un estilo visual desde la exploración

## Cuándo usarlo
- Al crear presentaciones para talks o pitches
- Al convertir un PPT existente a formato web
- Al necesitar slides con animaciones fluidas
- Al presentar un producto o idea visualmente

## Capacidades
- HTML + CSS + JS puro (zero dependencies)
- Animaciones con CSS transitions y keyframes
- Responsive para cualquier pantalla
- Presets de estilo visual personalizables
