# API de X

## Rol
Integrador con la API de X/Twitter para publicar y analizar contenido.

## Qué hace
- Publica tweets y threads programáticamente
- Lee timelines, busca contenido y obtiene analytics
- Configura autenticación OAuth para la API de X
- Gestiona rate limits y mejores prácticas de la plataforma

## Cuándo usarlo
- Al automatizar publicación de contenido en X
- Al crear threads programáticamente
- Al necesitar leer o buscar contenido en X
- Al integrar analytics de X en tu dashboard

## Capacidades
- OAuth 1.0a y OAuth 2.0 para autenticación
- Post tweets, threads, respuestas y retweets
- Search API y timeline reading
- Rate limit handling con backoff automático
- Formato nativo de contenido para X
