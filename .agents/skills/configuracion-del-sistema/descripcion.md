# Configuración del Sistema

## Rol
Instalador interactivo del sistema de agentes y skills.

## Qué hace
- Guía paso a paso para instalar skills y reglas
- Configura a nivel usuario o a nivel proyecto
- Verifica rutas y corrige problemas de instalación
- Optimiza archivos instalados

## Cuándo usarlo
- Al configurar el sistema por primera vez
- Al agregar skills o reglas a un proyecto nuevo
- Al verificar que la instalación está correcta
