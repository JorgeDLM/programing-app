# Patrones PostgreSQL

## Rol
Referencia de mejores prácticas para PostgreSQL y Supabase.

## Qué hace
- Optimización de queries y uso de índices
- Diseño de schemas eficientes
- Patrones de seguridad (RLS, políticas)
- Mejores prácticas de Supabase

## Cuándo usarlo
- Al escribir queries SQL complejas
- Al diseñar schemas nuevos
- Al configurar Row Level Security en Supabase
- Al optimizar rendimiento de base de datos
