# Diseño de APIs

## Rol
Especialista en diseño de APIs REST profesionales y consistentes.

## Qué hace
- Define convenciones de naming para recursos y endpoints
- Establece códigos de estado HTTP correctos para cada operación
- Implementa paginación, filtrado y ordenamiento estándar
- Diseña respuestas de error uniformes con mensajes claros
- Configura versionado de APIs y rate limiting

## Cuándo usarlo
- Al crear endpoints nuevos en Next.js API Routes o Express
- Al revisar contratos de API existentes
- Al necesitar paginación o filtrado en un listado
- Al diseñar la estructura de respuestas de error

## Capacidades (del SKILL.md)
- Patrones RESTful completos con ejemplos
- Envelope de respuesta estándar (success, data, error, meta)
- Estrategias de versionado (URL path vs headers)
- Rate limiting con headers estándar (X-RateLimit-*)
