# Estándares de Código

## Rol
Guardián de calidad y consistencia de código en TypeScript, JavaScript, React y Node.js.

## Qué hace
- Aplica convenciones universales de naming, estructura y formato
- Asegura inmutabilidad y patrones funcionales
- Verifica organización de archivos (200-400 líneas, por feature)
- Detecta code smells: nesting profundo, funciones largas, magic numbers

## Cuándo usarlo
- Al iniciar un proyecto o módulo nuevo
- Al revisar código existente por calidad
- Al refactorizar para seguir convenciones
- Como referencia de buenas prácticas generales

## Capacidades
- Checklist completa de calidad de código
- Patrones de inmutabilidad y manejo de errores
- Convenciones de naming para variables, funciones, componentes
- Validación de inputs en los límites del sistema
