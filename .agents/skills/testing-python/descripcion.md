# Testing Python

## Rol
Guía de testing para Python con pytest y TDD.

## Qué hace
- Estrategias de testing con pytest
- Fixtures, mocking y parametrización
- Cobertura mínima 80%
- Metodología TDD aplicada a Python

## Cuándo usarlo
- Al escribir tests para código Python
- Al configurar pytest en un proyecto nuevo
- Al implementar TDD en Python
- Al medir y mejorar cobertura de tests
