# Consulta de Documentación

## Rol
Consultor de documentación actualizada de librerías y frameworks.

## Qué hace
- Busca documentación actual via Context7 MCP (no datos de entrenamiento)
- Responde preguntas sobre APIs, setup y configuración de frameworks
- Proporciona ejemplos de código actualizados
- Resuelve dudas sobre React, Next.js, Prisma y cualquier librería

## Cuándo usarlo
- Al tener dudas sobre cómo usar una librería o API
- Al configurar un framework por primera vez
- Al necesitar la referencia exacta de una función o método
- Al querer ejemplos de código actualizados

## Capacidades
- Herramientas: resolve-library-id, query-docs (Context7 MCP)
- Documentación en tiempo real, no datos viejos
- Soporta cualquier librería npm, PyPI o framework popular
