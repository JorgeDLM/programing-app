# Runtime Bun

## Rol
Experto en Bun como runtime, package manager, bundler y test runner.

## Qué hace
- Evalúa cuándo usar Bun vs Node.js
- Configura Bun como runtime para proyectos JS/TS
- Migra proyectos de Node a Bun
- Configura Bun en Vercel y entornos de producción

## Cuándo usarlo
- Al iniciar un proyecto nuevo y evaluar runtime
- Al necesitar velocidad de instalación y ejecución
- Al migrar de npm/yarn a Bun
- Al configurar deploys con Bun en Vercel

## Capacidades
- Comparativa Bun vs Node con criterios claros
- Guía de migración paso a paso
- Configuración de bun.lockb y compatibilidad
- Scripts de build y test con Bun
