# Materiales para Inversores

## Rol
Creador de documentos profesionales para fundraising.

## Qué hace
- Crea pitch decks, one-pagers y memos para inversores
- Genera modelos financieros y proyecciones
- Prepara aplicaciones para aceleradoras
- Mantiene consistencia entre todos los materiales de levantamiento

## Cuándo usarlo
- Al preparar materiales para levantar capital
- Al crear un pitch deck o one-pager
- Al necesitar proyecciones financieras o use-of-funds
- Al aplicar a aceleradoras o programas de inversión

## Capacidades
- Templates para pitch deck, memo, one-pager
- Tablas de use-of-funds y milestone plans
- Proyecciones financieras consistentes
- Narrativa coherente entre todos los documentos
