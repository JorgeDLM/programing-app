# Inventario de Skills

## Rol
Auditor de calidad de skills y commands del sistema.

## Qué hace
- Audita todos los skills con checklist de calidad
- Modo Quick Scan (solo cambios recientes) y Full Stocktake (completo)
- Evalúa con IA para detectar problemas de calidad
- Genera reportes de estado por skill

## Cuándo usarlo
- Al hacer mantenimiento periódico del sistema de skills
- Al querer evaluar la calidad de los skills existentes
- Después de agregar o modificar skills
