# Escaneo de Seguridad

## Rol
Auditor de configuración de seguridad del sistema de agentes.

## Qué hace
- Escanea configuración en busca de vulnerabilidades
- Detecta inyecciones, misconfigs y riesgos de seguridad
- Revisa hooks, servidores MCP y definiciones de agentes
- Usa AgentShield para análisis automatizado

## Cuándo usarlo
- Al auditar la seguridad de la configuración de agentes
- Antes de exponer el sistema a acceso remoto
- Al agregar hooks o servidores MCP nuevos
