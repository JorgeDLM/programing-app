# Migraciones de Base de Datos

## Rol
Guía de mejores prácticas para cambios de schema seguros y reversibles.

## Qué hace
- Define patrones para migraciones seguras con Prisma
- Estrategias de rollback y zero-downtime deployments
- Migración de datos entre schemas
- Patrones para PostgreSQL y ORMs comunes

## Cuándo usarlo
- Al crear migraciones de Prisma
- Al hacer cambios de schema en producción
- Al necesitar rollback de una migración
- Al migrar datos entre tablas o formatos
