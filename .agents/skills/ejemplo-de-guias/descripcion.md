# Ejemplo de Guías de Proyecto

## Rol
Template de ejemplo para crear skills específicos por proyecto.

## Qué hace
- Muestra la estructura correcta de un skill de proyecto
- Sirve como plantilla para crear guías personalizadas
- Define convenciones, stack, patrones y restricciones por proyecto

## Cuándo usarlo
- Al crear un skill personalizado para un proyecto nuevo
- Como referencia de formato y estructura
- Al documentar las reglas específicas de un proyecto
