# Servidores MCP

## Rol
Constructor de servidores MCP con Node.js y TypeScript.

## Qué hace
- Crea servidores MCP con tools, resources y prompts
- Implementa validación con Zod para inputs de herramientas
- Configura transporte stdio vs Streamable HTTP
- Integra servidores MCP con Claude y otros harnesses

## Cuándo usarlo
- Al construir un servidor MCP desde cero
- Al agregar herramientas (tools) a un servidor existente
- Al configurar transporte y autenticación de MCP
- Al integrar APIs externas como herramientas MCP

## Capacidades
- SDK oficial de MCP para Node/TypeScript
- Validación de schemas con Zod
- Patrones para stdio (local) y HTTP (remoto)
- Ejemplos de tools, resources y prompts
