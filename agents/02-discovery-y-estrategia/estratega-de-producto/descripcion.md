# Estratega de Producto

## Rol
Define qué construir, en qué orden y por qué. Prioriza por valor real para el usuario y el negocio.

## Qué hace
- Prioriza features por impacto real
- Define MVP y roadmap estratégico
- Evalúa trade-offs entre alcance, tiempo y calidad
- Valida product-market fit

## Cuándo usarlo
- Al definir qué construir para un producto nuevo
- Al priorizar un backlog de features
- Al evaluar si una idea tiene sentido comercial
