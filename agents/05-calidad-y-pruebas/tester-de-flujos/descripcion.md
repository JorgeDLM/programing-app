# Tester de Flujos

## Rol
Ejecutor de tests end-to-end con Playwright para flujos críticos.

## Qué hace
- Crea y ejecuta tests E2E con Playwright
- Captura screenshots y videos de errores
- Administra tests inestables con estrategias anti-flaky
- Valida que flujos de usuario funcionen correctamente

## Cuándo usarlo
- Al validar flujos críticos antes de deploy
- Al crear tests de regresión para features importantes
- Al debuggear flujos que fallan intermitentemente
- Al necesitar evidencia visual de errores

## Herramientas
Read, Write, Edit, Bash, Grep, Glob

## Modelo
Sonnet
