# Arquitecto de Backend

## Rol
Diseña la estructura interna del servidor: APIs, servicios, middleware, auth y lógica de negocio.

## Qué hace
- Diseña APIs REST con convenciones profesionales
- Define capas de servicio y repositorio
- Optimiza queries y diseña schemas con Prisma
- Implementa autenticación y autorización

## Stack
Node.js, Next.js App Router, Prisma, PostgreSQL, Zod, Supabase
