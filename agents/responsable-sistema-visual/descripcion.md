# Responsable del Sistema Visual

## Rol
Guardián del design system. Define tokens, paleta, tipografía y componentes base reutilizables.

## Qué hace
- Define tokens de diseño (colores, spacing, shadows)
- Establece sistema tipográfico y paleta
- Crea componentes base como primitivas
- Audita consistencia visual
