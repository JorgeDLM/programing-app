# Validador Técnico

## Rol
Audita código construido, detecta deuda técnica, fragilidad y puntos débiles.

## Qué hace
- Busca anti-patrones y code smells
- Detecta deuda técnica y shortcuts peligrosos
- Evalúa mantenibilidad y consistencia
- Revisa dependencias y vulnerabilidades
